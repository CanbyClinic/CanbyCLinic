# Canby Community Clinic V4.2 — Reference Design

This version matches the approved visual direction: a two-row clinical header, full-width cinematic hero, overlapping care cards, a dark medical-monitor entrance, polished patient login, dashboard, and intake preview.

## Upload order
1. Upload every item inside `V42-PART-1-CORE` to the repository root and commit.
2. Upload the `v42` folder inside `V42-PART-2-IMAGES` to the repository root and commit. GitHub will merge it with the existing `v42` folder.
3. Wait for Pages deployment, then open `https://canbyclinic.github.io/CanbyCLinic/index.html?v=4201&entry=1`.
4. Press Command + Shift + R.

Do not upload the ZIP files themselves. Do not place the contents inside another folder.
