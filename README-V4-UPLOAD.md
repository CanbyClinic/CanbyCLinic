# Canby Community Clinic V4 — Upload in Two Parts

V4 balances a premium editorial experience with the clarity and trust expected from a healthcare organization.

## Upload order
1. Upload everything inside **V4-PART-1-CODE** to the root of the GitHub repository. Commit.
2. Upload everything inside **V4-PART-2-ASSETS** to the same repository root. The `assets` folders will merge. Commit.
3. Wait for GitHub Pages to deploy, then hard refresh with Command + Shift + R.

## Portal preview
The portal is a front-end prototype. Account and intake demo data are stored only in the visitor's browser using local storage. It is not transmitted to the clinic and must not be used for real medical information. Connect a compliant backend before production use.

## Main V4 additions
- detailed heartbeat monitor entrance animation
- large editorial homepage imagery with exact aspect ratios
- patient sign-up, sign-in, password recovery preview, remembered login, and dashboard
- dashboard sections for appointments, notes, prescriptions, results, documents, messages, and profile
- first-page patient demographics intake in English and Spanish
- new 2025 community-health research review
- Journal link in the global navigation
